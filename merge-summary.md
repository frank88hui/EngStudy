此次合并引入了一个完整的 521词小学英语口语学习应用，包含单词学习、分类浏览、发音功能、学习进度跟踪和成就系统等核心功能。应用使用 React 构建，采用儿童友好的界面设计，支持响应式布局，适合小学生使用。
| 文件 | 变更 |
|------|---------|
| src/App.jsx | - 新增主应用组件，包含路由配置和页面布局<br>- 实现了加载动画和页面导航功能 |
| src/WordLearning.jsx | - 新增单词学习组件，实现了单词卡片翻转、发音、收藏、标记已学习和添加到生词本功能<br>- 集成了学习进度跟踪、成就系统和奖励通知功能<br>- 实现了分类选择和单词导航功能 |
| src/data.js | - 新增单词数据文件，包含人物、动物、水果、食物饮品和颜色五个分类的单词<br>- 每个单词包含英文、中文含义、例句和音标 |
| src/index.css | - 新增样式文件，采用儿童友好的色彩方案和字体<br>- 实现了响应式布局和动画效果 |
| src/main.jsx | - 新增应用入口文件，配置了 React 根组件和路由 |
| vite.config.js | - 新增 Vite 配置文件，配置了 React 插件 |
| package.json | - 新增项目依赖配置，包含 React、React Router、Vite 等依赖 |
| package-lock.json | - 新增依赖锁文件，确保依赖版本一致性 |
