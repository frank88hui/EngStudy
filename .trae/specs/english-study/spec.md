# 521词小学英语口语 - 产品需求文档

## Overview
- **Summary**: 一个基于React的小学英语口语学习应用，用于帮助小学生学习521个英语单词，包括单词展示、含义解释和例句展示功能。
- **Purpose**: 解决小学生英语单词学习的问题，提供一个简洁、直观的学习平台，帮助学生掌握基础英语单词。
- **Target Users**: 小学生（6-12岁）及其家长、教师。

## Goals
- 提供521个英语单词的学习功能
- 展示单词的含义和例句
- 提供单词导航功能（上一个/下一个）
- 支持明暗模式切换
- 提供美观、易用的用户界面

## Non-Goals (Out of Scope)
- 不包含语音发音功能
- 不包含单词测试功能
- 不包含用户账户系统
- 不包含数据持久化存储
- 不包含多语言支持（仅中文界面）

## Background & Context
- 项目基于React和Vite构建，使用现代前端技术
- 单词数据目前为模拟数据，后续可从Excel文件导入完整的521个单词
- 应用设计考虑了小学生的使用习惯，界面简洁直观

## Functional Requirements
- **FR-1**: 单词展示功能 - 显示单词的拼写
- **FR-2**: 含义展示功能 - 显示单词的中文含义
- **FR-3**: 例句展示功能 - 显示包含该单词的例句
- **FR-4**: 单词导航功能 - 支持上一个/下一个单词切换
- **FR-5**: 含义显示控制 - 支持显示/隐藏单词含义和例句

## Non-Functional Requirements
- **NFR-1**: 响应式设计 - 适配不同屏幕尺寸
- **NFR-2**: 性能要求 - 页面加载和操作响应时间不超过1秒
- **NFR-3**: 可访问性 - 符合Web内容可访问性指南(WCAG) 2.1标准
- **NFR-4**: 视觉设计 - 美观、现代，适合小学生使用

## Constraints
- **Technical**: React 18+, Vite 5+, 现代浏览器兼容性
- **Business**: 无特殊业务约束
- **Dependencies**: React, React DOM, Vite

## Assumptions
- 单词数据将从Excel文件导入，目前使用模拟数据
- 用户使用现代浏览器访问应用
- 应用运行在本地开发环境或部署到静态网站托管服务

## Acceptance Criteria

### AC-1: 单词展示
- **Given**: 用户打开应用
- **When**: 应用加载完成
- **Then**: 显示第一个单词的拼写
- **Verification**: `human-judgment`
- **Notes**: 单词应清晰显示，字体大小适中

### AC-2: 含义和例句展示
- **Given**: 用户点击"显示含义"按钮
- **When**: 按钮被点击
- **Then**: 显示单词的中文含义和例句
- **Verification**: `human-judgment`
- **Notes**: 含义和例句应清晰可读

### AC-3: 单词导航
- **Given**: 用户点击"上一个"或"下一个"按钮
- **When**: 导航按钮被点击
- **Then**: 显示相应的单词
- **Verification**: `human-judgment`
- **Notes**: 导航应流畅，单词切换无明显延迟

### AC-4: 响应式设计
- **Given**: 用户在不同设备上访问应用
- **When**: 屏幕尺寸变化
- **Then**: 应用界面自适应调整
- **Verification**: `human-judgment`
- **Notes**: 在手机、平板和桌面设备上均应正常显示

### AC-5: 明暗模式支持
- **Given**: 用户系统设置为明暗模式
- **When**: 应用加载
- **Then**: 应用自动适配系统的明暗模式设置
- **Verification**: `human-judgment`
- **Notes**: 颜色方案应在两种模式下都清晰可读
